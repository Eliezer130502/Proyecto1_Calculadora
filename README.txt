PROYECTO 1 CALCULADORA: 

Este proyecto es una aplicación de calculadora con un backend en Spring Boot y un frontend en React. A continuación se detallan los pasos para ejecutar el proyecto.

REQUISITOS
Herramientas que se deben instalar para el funcionamiento del proyecto:
-	Java 21: Necesario para ejecutar el backend.
-	Maven: Herramienta de gestión de proyectos para Java.
-	Node.js y npm: Necesarios para ejecutar el frontend.

ESTRUCTURA DEL PROYECTO 
-	Backend: calculadora_backend (Java con Spring Boot)
-	Frontend: calculadora-frontend (React)

PASOS A SEGUIR PARA LA EJECUCIÓN
1.	Ejecución del backend desde CMD
Navegar al directorio: cd C:\Users\emanu\Desktop\Proyecto1_Calculadora\calculadora_backend
Ejecutar: mvn spring-boot:run
-	Para verlo en el navegador el servidor es el siguiente: http://localhost:8080
-	Para ver el historial: http://localhost:8080/api/history
2.	Ejecución del frontend dede CMD
Navegar al directorio: cd C:\Users\emanu\Desktop\Proyecto1_Calculadora\calculadora-frontend
Instalar las dependencias: npm install
Ejecutar: npm start
-	Para verlo en el navegador el servidor es el siguiente: http://localhost:3000
